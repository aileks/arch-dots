# Dotfiles

- i3
- i3lock-fancy
- i3blocks
- dmenu
- rofi
- maim
- eza
- neovim
- tmux
- feh
- picom
- zsh
- thunar
- dunst
- xclip
- yad
- playerctl
- trash-cli
